# Vibes Preinstalled Marketplace

Pre-installed MCP-based plugins that ship with Vibespace workspaces.

## Plugins

### vibes-n8n

n8n workflow management tools for Claude Code.

**MCP Tools:**
- `list_workflows` - List all workflows in an n8n project
- `get_workflow` - Get workflow JSON by ID or name
- `sync_workflows` - Sync workflows from remote n8n instance
- `push_workflow` - Push local workflow changes to n8n

**Commands:**
- `/n8n-sync` - Sync all workflows from remote n8n instance
- `/n8n-push` - Push modified workflow to remote
- `/n8n-list` - List all workflows with status

**Agents:**
- `workflow-analyzer` - Analyze workflow structure and suggest optimizations
- `workflow-builder` - Guide through creating new workflows

## Installation

This marketplace is automatically installed in Vibespace workspaces during setup.
