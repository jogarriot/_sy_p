# workspace

Workspace management plugin for Vibespace workspaces - projects, integrations, and mini apps.

## MCP Tools

This plugin provides workspace management tools via the `workspace-mcp` server:

| Tool | Description |
|------|-------------|
| `projects` | Manage projects (list, add, remove, update, details, refresh) |
| `integrations` | Manage integrations (list, add, remove, update, enable, disable, details) |
| `mini_apps` | Manage mini apps/shortcuts (list, add, remove, update, start, stop, details) |
| `browser` | Interact with browser tabs (get logs, network requests, navigate, run JavaScript) |
